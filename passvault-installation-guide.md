# How to Install PassVault

Before you begin, please have your mobile device nearby to complete the required security authentication.

## Installation Steps

1. Go to the company portal page: `://company.com`.
2. Download the installer for your operating system:
   * Mac users: **PassVault_Setup.dmg**
   * Windows users: **PassVault_Setup.exe**
3. Run the downloaded file. When prompted for a **Server Address**, type exactly: `://company.com`
4. Log in using your standard corporate email address.
5. Retrieve the two-factor authentication (2FA) code sent to your mobile phone and type it into the app to finalize the setup.
